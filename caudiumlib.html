HTTP Return Functions

The following functions are used to generate specific HTTP result codes, and can be accessed by inheriting http. Note that http is inherited by caudiumlib, so it is sufficient to inherit that to gain access to these functions.

These functions, often called "http mapping answer functions" (???) return mappings, and may be used in any script or module function that can return this type of mapping.

http_auth_required()
http_error_answer()
http_file_answer()
http_htmldoc_answer()
http_low_answer()
http_pipe_in_progress()
http_proxy_auth_required()
http_redirect()
http_rxml_answer()
http_stream()
http_string_answer()
